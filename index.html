Con CursOn pasarás al siguiente nivel!

Nuestros cursos en línea son los mejores del mercado.

Aprende donde sea, acompañado y a tu ritmo!

Destacados

HTML para principiantes

Aprenderás los fundamentos de HTML, el lenguaje base de toda página web, de forma clara y sencilla. Con explicaciones prácticas y ejercicios, te guiaremos desde la estructura básica de un sitio web hasta la creación de tu propio proyecto.

CSS diseño responsive

En este curso aprenderás cómo darle vida y estilo a tus páginas web utilizando CSS, el lenguaje que permite transformar un simple sitio en algo visualmente atractivo.

JavaScript fundamentos

Aprenderás los conceptos fundamentales de forma clara y práctica: desde variables y funciones hasta la manipulación del DOM para hacer tus páginas dinámicas. No necesitas experiencia previa, solo curiosidad y ganas de desarrollar habilidades para crear webs interactivas y divertidas.

Lo nuevo

React de cero a experto

Este curso te llevará de la mano en el aprendizaje de React, la biblioteca más popular para construir interfaces web modernas y dinámicas. Aprenderás los conceptos clave como componentes, estados y props, y cómo crear aplicaciones interactivas de manera eficiente.

VueJS para principiantes

En este curso aprenderás a usar Vue.js, un framework intuitivo y flexible que te permitirá crear aplicaciones web interactivas y dinámicas con facilidad. Desde conceptos básicos como componentes y directivas, hasta la gestión de estados y eventos, te guiaremos paso a paso para que puedas desarrollar tus propios proyectos.

Angular para expertos

En este curso aprenderás a utilizar Angular, uno de los frameworks más poderosos y versátiles para desarrollar aplicaciones web robustas y escalables. Desde los fundamentos como componentes y módulos, hasta la gestión de rutas y servicios, te guiaremos paso a paso para que puedas crear proyectos dinámicos y profesionales.

NodeJS intermedio

En este curso aprenderás a utilizar Node.js, el entorno que te permitirá crear aplicaciones del lado del servidor utilizando JavaScript. Desde los fundamentos como módulos y eventos, hasta la creación de APIs y manejo de bases de datos, te guiaremos paso a paso para desarrollar aplicaciones backend modernas y eficientes.
